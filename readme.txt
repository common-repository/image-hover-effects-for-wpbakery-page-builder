Creative hover images effects addon for WPBakery Page Builder (Visual Composer)

== Description ==

Image hover effects addon for WPBakery Page Builder lets you add hover image to your website with various options.

There are different options to set such as image, box height, set hover content as default, set title, button with text size and color.

Moreover it is one of the best hover image plugin for developers , it is developed with WordPress actions, so developer can modify it in theme according to the client's need.

== Installation ==

1. Upload the plugin folder `image-hover-effects-addon` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins -> Installed plugins -> Activate Image hover Effects Addon" menu in WordPress.
3. That's it. Now you can find the 'Image hover box' element in visual composer element list.
